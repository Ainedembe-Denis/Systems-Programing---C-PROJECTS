# 2. EDUCATION PROJECTS
This folder contains C programs designed for educational environments and student data management.

## Projects

### 1. Library Management System
- **File**: `Library-Management-System/Library-Management-System.c`
- **Description**: A system to track library inventory and book circulation.
- **Key Features**:
  - Add and search for books by ID.
  - Issue books to students with specified return dates.
  - Automatic timestamping of issuance.
  - Status tracking (Available/Issued).

### 2. Student Record Management System
- **File**: `Student-Record-Management-System/Student-Record-Mgt-System.c`
- **Description**: A tool for teachers and administrators to manage student grades and performance.
- **Key Features**:
  - Store student details and marks for multiple subjects.
  - Calculate average marks automatically.
  - Assign grades based on performance.
  - Generate class-wide performance reports.
