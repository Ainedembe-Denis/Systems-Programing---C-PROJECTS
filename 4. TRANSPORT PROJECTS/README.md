# 4. TRANSPORT PROJECTS
This folder contains C programs designed for the transportation industry and fleet logistics.

## Projects

### 1. Bus Reservation System
- **File**: `Bus-Reservation-System/Bus-Reservation-System.c`
- **Description**: A visual seat booking system for commercial buses.
- **Key Features**:
  - Interactive seat layout display (Available/Booked).
  - Booking and cancellation of specific seats.
  - Persistent storage of bookings in binary files.
  - Passenger list generation.

### 2. Vehicle Fleet Management System
- **File**: `Vehicle-Fleet-Management-System/Vehicle-Fleet-Management-System.c`
- **Description**: A system to track the health, usage, and efficiency of a vehicle fleet.
- **Key Features**:
  - Record and track vehicle mileage and fuel consumption.
  - Log maintenance activities.
  - Automatic maintenance alerts based on mileage thresholds.
  - Detailed fleet performance reports.
