# 3. MEDICAL PROJECTS
This folder contains C programs developed for healthcare facilities and pharmacy management.

## Projects

### 1. Hospital Patient Management System
- **File**: `Hospital-Patient-Management-System/Hospital-Patient-Management-System.c`
- **Description**: A comprehensive system to manage hospital operations, including personnel, patients, and financial records.
- **Key Features**:
  - Manage doctor and patient databases.
  - Schedule and view appointments with status tracking.
  - Record diagnoses and generate medical bills.
  - View full patient history across multiple visits.

### 2. Pharmacy Inventory System
- **File**: `Pharmacy-Inventory-System/Pharmacy-Inventory-System.c`
- **Description**: A tool for managing medicine stock and recording pharmaceutical sales.
- **Key Features**:
  - Track medicine inventory, prices, and expiry dates.
  - Record sales and automatically update stock levels.
  - Add new medicines to the database.
  - Generate stock reports.
