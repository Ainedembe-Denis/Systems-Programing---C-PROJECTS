# Systems Programming - C Projects

A collection of 8 practical C programming projects organized into four major categories. This repository demonstrates core systems programming concepts like file I/O, binary data management, security/authentication, and mathematical modeling in C.

## Repository Structure

### [1. BANKING PROJECTS](./1.%20%20BANKING%20PROJECTS/)
- **Bank Account Management System**: Secure account management with audit logging.
- **Loan Eligibility & EMI Calculator**: Financial eligibility and payment modeling.

### [2. EDUCATION PROJECTS](./2.%20EDUCATION%20PROJECTS/)
- **Library Management System**: Inventory and circulation tracking for books.
- **Student Record Management System**: Academic data management and grading.

### [3. MEDICAL PROJECTS](./3.%20MEDICAL%20PROJECTS/)
- **Hospital Patient Management System**: Healthcare workflow and billing system.
- **Pharmacy Inventory System**: Stock control and pharmaceutical sales tracking.

### [4. TRANSPORT PROJECTS](./4.%20TRANSPORT%20PROJECTS/)
- **Bus Reservation System**: Interactive booking system with seat visualization.
- **Vehicle Fleet Management System**: Logistics and maintenance performance tracking.

## Key Technical Concepts
- **File Systems**: Uses `.dat` binary files for persistent data storage.
- **Structures**: Custom `struct` definitions for complex data modeling.
- **Memory Management**: Handled via standard library and local allocation.
- **Time/Logging**: Integrated `time.h` for audit trails and timestamps.
