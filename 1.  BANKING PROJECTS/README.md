# 1. BANKING PROJECTS
This folder contains C programs related to banking and financial management.

## Projects

### 1. Bank Account Management System
- **File**: `Bank-Account-Management-System/Bank-Account-Management-System.c`
- **Description**: A comprehensive system for managing bank accounts. It supports account creation, deposits, withdrawals, and balance inquiries. It includes a login system and audit logging for security.
- **Key Features**:
  - Role-based access (Teller/Admin).
  - Multi-currency support (UGX/USD).
  - Minimum balance enforcement.
  - Append-only audit logging.

### 2. Loan Eligibility and EMI Calculator
- **File**: `Loan-Eligibility-and-EMI-Calculator/Loan-Eligibility-and-EMI-Calculator.c`
- **Description**: A tool to help users determine their eligibility for a loan and calculate their monthly payments.
- **Key Features**:
  - EMI calculation using the standard amortization formula.
  - Eligibility check based on debt-to-income ratio (40% threshold).
  - User-friendly input for salary, loan amount, and interest rates.
